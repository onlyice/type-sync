# 使用指南

## 快速开始

1. **启动 Mac 服务端**
   ```bash
   cd mac-server && .build/debug/TypeSyncServer
   ```
   菜单栏出现 ⌨️ 图标，点击可查看 IP 地址和连接状态。

2. **打开 Android 客户端**
   - 确保手机与 Mac 在同一局域网
   - 如果 mDNS 发现成功，直接点击设备名称自动连接
   - 或手动输入 Mac 的 IP 地址，端口 `9876`，点击 **连接**
   - 顶部状态栏显示 🟢 表示连接成功

## 功能说明

### 文本输入

主界面的文本输入框支持：
- **键盘输入** — 输入的字符实时同步到 Mac 的当前焦点应用
- **语音输入** — 使用 Android 键盘的语音输入按钮，语音识别后自动同步到 Mac
- **清空输入框** — 只清空手机端输入框，不影响 Mac 端

输入框下方的剪贴板按钮：
- **📋→Mac** — 将手机剪贴板内容推送到 Mac 剪贴板
- **Mac→📋** — 拉取 Mac 剪贴板内容到手机，显示后可复制

## 网络要求

- Mac 和 Android 设备须在**同一局域网**
- 服务端监听端口 **9876**（TCP WebSocket）
- 使用明文 `ws://` 连接（非加密）
- 如果使用 VPN（如 Tailscale），可能需要使用 VPN 分配的 IP 或确保局域网直连可用

## 故障排除

| 问题 | 解决方案 |
|------|---------|
| 连接失败 | 检查同一局域网、防火墙未阻止 9876 端口 |
| 按键无反应 | 检查 Mac 是否授予辅助功能权限 |
| mDNS 未发现设备 | 手动输入 IP 连接；确认路由器未隔离设备 |
| 连接频繁断开 | 检查 Wi-Fi 信号；客户端有自动重连（最多 10 次） |
